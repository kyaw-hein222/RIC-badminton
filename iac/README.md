# Infrastructure as Code

This folder is the home for the RIC Badminton Terraform configuration.

We will add the configuration gradually as part of the Terraform learning path:

1. Provider and version configuration
2. Terraform state storage
3. DynamoDB
4. Lambda and IAM
5. API Gateway
6. S3 and CloudFront

Do not store AWS credentials, passcodes, or `terraform.tfstate` files in this repository.

The old `infrastructure/` folder was intentionally removed so there is one clear IaC location.
